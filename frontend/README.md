# Power Monitoring Visualization Tool

Client-side web application for interactive visualization and analysis of residential power consumption data.

## Purpose

Visualize power consumption patterns to help electricians:
- Identify circuit breaker trip causes
- Compare weekday vs weekend usage
- Analyze time-of-day patterns
- Filter data by date range, time, and room

## Requirements

- Modern web browser (Chrome, Firefox, or Edge - latest version)
- Generated CSV file from data-generator tool

## Setup

No installation required! This is a pure client-side application.

### Option 1: Open Directly

1. Open `frontend/index.html` in your browser
2. Click "Load CSV File" and select your power data file
3. Charts will render automatically

### Option 2: Local Web Server (Recommended for Development)

```bash
# Python 3
cd frontend
python -m http.server 8000

# Then open: http://localhost:8000
```

## Usage

### 1. Load Data

- Click **"Load CSV File"** button
- Select CSV file generated by the data-generator tool
- Data loads and parses (typically < 1 second)

### 2. View Time Series

- **Interactive charts** show power consumption over time
- **Color-coded lines**: One per room (colorblind-safe palette)
- **Red markers**: Indicate breaker trips
- **Tooltips**: Hover for timestamp, wattage, amperage details

### 3. Zoom and Pan

- **Zoom**: Click and drag horizontally on the chart
- **Pan**: After zooming, drag to move view window
- **Reset**: Double-click chart or click "Reset Zoom" button

### 4. Compare Weekday vs Weekend

- Click **"Comparison View"** toggle
- View side-by-side or overlay of weekday vs weekend patterns
- See statistics showing peak differences during critical hours (3-4pm)

### 5. Filter Data

- **Date Range**: Select start and end dates
- **Time of Day**: Filter to specific hour ranges (e.g., 3-4pm)
- **Rooms**: Show/hide specific rooms via checkboxes
- **Reset**: Click "Clear Filters" to restore full dataset

### 6. Export Results

- **PNG**: Export current chart view as image
- **PDF**: Generate report with charts and summary statistics
- **CSV**: Export filtered dataset for further analysis

## Troubleshooting

### Charts Not Displaying
- Check browser console for errors
- Verify CSV file format matches specification
- Ensure Highcharts library is loaded (check Network tab)

### Slow Performance
- Reduce date range via filters
- Disable some rooms to reduce data points
- Use WebGL-accelerated browser (Chrome recommended)

### CSV Load Errors
- Verify CSV has required columns: timestamp, room_name, wattage, amperage, voltage, breaker_tripped
- Check for negative wattage values (invalid)
- Ensure timestamps are in ISO 8601 format

## Development

### Code Structure

- **ES6 Modules**: Native browser modules (no build step required)
- **Event Bus Pattern**: Decoupled communication between modules
- **PapaParse**: CSV parsing with Web Worker support
- **Highcharts + Boost**: High-performance charting with WebGL acceleration

### Adding Features

1. Create new module in `js/` directory
2. Import event bus: `import { eventBus } from './core/eventBus.js'`
3. Subscribe to relevant events
4. Emit new events for other modules to consume
5. Update `main.js` to initialize new module
